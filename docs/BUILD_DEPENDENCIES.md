# Tsukika (月華) | Build Dependencies

- java
- openssl (ig if u have openjdk)
- python3
- zip
- lz4
- tar
- file
- unzip
- simg2img
- make
- xmlstarlet
- mkfs.erofs
- mkdtimg
- imjtool

> ⚠️ **WARNING:** Just know that the build will fail if the required dependencies are not available on your host.
